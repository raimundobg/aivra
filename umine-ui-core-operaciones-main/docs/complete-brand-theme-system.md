# Complete Brand Theme System - Technical Specification

## Executive Summary

This document defines the **complete semantic token structure** required for a fully functional brand identity system in the Umine Design System. Every color category displayed in `/colors` must be generated by the Lambda AI and properly mapped in the theme provider.

---

## Problem Statement

Currently, the brand theme system has **incomplete coverage**:

1. **Foregrounds (Text)**: Only `fg.brand` is mapped from brand colors. Missing: `fg.default`, `fg.muted`, `fg.accent`, `fg.inverted`
2. **System Buttons**: Only `button.primary` uses brand colors. Missing: hover states, secondary button variants
3. **Tags/Badges**: Hardcoded to system palettes (`green`, `blue`, `orange`, `red`, `purple`, `cyan`). Not brand-aware.
4. **Raw Palette**: Shows hardcoded `green`, `cyan`, `slate` scales. Should show brand-generated color scales.
5. **UI Elements**: "Evolve with AI" button and Preview components don't use brand colors.

---

## Complete Token Structure

### 1. **Palette (Raw Colors)**

The Lambda must generate a **complete color palette** with scales:

```json
{
  "palette": {
    "primary": "#hex",
    "secondary": "#hex",
    "accent": "#hex",
    "neutrals": ["#hex1", "#hex2", "#hex3", "#hex4", "#hex5"],
    "scales": {
      "primary": {
        "50": "#hex",
        "100": "#hex",
        "200": "#hex",
        "300": "#hex",
        "400": "#hex",
        "500": "#hex (base primary)",
        "600": "#hex",
        "700": "#hex",
        "800": "#hex",
        "900": "#hex"
      },
      "secondary": {
        "50": "#hex",
        // ... same structure
        "900": "#hex"
      },
      "accent": {
        "50": "#hex",
        // ... same structure
        "900": "#hex"
      }
    }
  }
}
```

**Purpose**: These scales will replace the hardcoded `green`, `cyan`, `slate` in the "Raw Palette" section.

### 2. **Semantic Tokens (Complete Structure)**

The Lambda must generate **all** semantic tokens required by the UI:

```json
{
  "semanticTokens": {
    "colors": {
      // BRAND CORE
      "brand": {
        "primary": { "value": "{palette.primary}" },
        "secondary": { "value": "{palette.secondary}" },
        "accent": { "value": "{palette.accent}" }
      },
      
      // FOREGROUNDS (TEXT)
      "fg": {
        "default": { 
          "value": { 
            "_light": "{palette.scales.primary.800}", 
            "_dark": "{palette.neutrals[0]}" 
          } 
        },
        "muted": { 
          "value": { 
            "_light": "{palette.scales.primary.500}", 
            "_dark": "{palette.scales.primary.400}" 
          } 
        },
        "accent": { 
          "value": { 
            "_light": "{palette.accent}", 
            "_dark": "{palette.scales.accent.400}" 
          } 
        },
        "inverted": { 
          "value": { 
            "_light": "white", 
            "_dark": "{palette.scales.primary.900}" 
          } 
        },
        "brand": { 
          "value": "{palette.primary}" 
        }
      },
      
      // BACKGROUNDS
      "bg": {
        "canvas": { 
          "value": { 
            "_light": "{palette.neutrals[0]}", 
            "_dark": "{palette.scales.primary.950}" 
          } 
        },
        "surface": { 
          "value": { 
            "_light": "rgba(255, 255, 255, 0.7)", 
            "_dark": "rgba({palette.scales.primary.900}, 0.6)" 
          } 
        },
        "muted": { 
          "value": { 
            "_light": "{palette.neutrals[1]}", 
            "_dark": "{palette.scales.primary.900}" 
          } 
        },
        "brand": { 
          "value": "{palette.primary}" 
        },
        "accent": { 
          "value": "{palette.accent}" 
        }
      },
      
      // BORDERS
      "border": {
        "subtle": { 
          "value": { 
            "_light": "rgba({palette.primary}, 0.08)", 
            "_dark": "rgba({palette.neutrals[0]}, 0.12)" 
          } 
        },
        "brand": { 
          "value": "{palette.primary}" 
        },
        "accent": { 
          "value": "{palette.accent}" 
        }
      },
      
      // BUTTONS
      "button": {
        "primary": {
          "bg": { "value": "{palette.primary}" },
          "hover": { "value": "{palette.scales.primary.600}" },
          "fg": { "value": "#ffffff or #000000 (based on contrast)" }
        },
        "secondary": {
          "bg": { "value": "transparent" },
          "hover": { "value": "{palette.scales.secondary.500}/10" },
          "fg": { "value": "{palette.secondary}" },
          "border": { "value": "{palette.secondary}" }
        },
        "accent": {
          "bg": { "value": "{palette.accent}" },
          "hover": { "value": "{palette.scales.accent.600}" },
          "fg": { "value": "#ffffff or #000000 (based on contrast)" }
        }
      },
      
      // STATUS COLORS (for badges/tags)
      "status": {
        "success": { 
          "value": { 
            "_light": "{palette.scales.primary.500}", 
            "_dark": "{palette.scales.primary.400}" 
          } 
        },
        "info": { 
          "value": { 
            "_light": "{palette.scales.accent.500}", 
            "_dark": "{palette.scales.accent.400}" 
          } 
        },
        "warning": { 
          "value": { 
            "_light": "{palette.scales.secondary.500}", 
            "_dark": "{palette.scales.secondary.400}" 
          } 
        },
        "error": { 
          "value": { 
            "_light": "#ef4444", 
            "_dark": "#f87171" 
          } 
        }
      }
    }
  }
}
```

```typescript

    brandSecondary: overlay.palette.scales.secondary,
    brandAccent: overlay.palette.scales.accent

}
```

## Visual Coherence Requirements

### 1. **Internal Solutions (Umine Apps)**
- Must use the generated brand identity consistently
- All buttons, badges, and interactive elements reflect the brand
- Color palette is fully derived from the brand configuration

### 2. **External Solutions (Client Apps)**
- Clients can generate their own brand identity
- The system adapts 100% to the client's palette
- No hardcoded Umine colors should appear in client-facing apps

### 3. **Template Themes**
- Mock brands (Lunar Banking, Solar Energy, etc.) serve as templates
- Each template has a complete palette with scales
- Templates demonstrate the full range of the system's capabilities
